##VACATION
This code can be used to calculate the maximum holiday span for the given vacation days.

##Prerequisites
Following python libraries:
datetime
holidays
numpy
tkinter

##Usage 
Run "gui_for_vacation.py", enter the name of the 'Country' you live in , 'Year' and number of vacation days.
Then press 'Calculate' button in the GUI window. You will get the possible vacation dates for maximum vacation span on the output window.

##Example
Country Name = "Germany"
Year = 2020
Number of Vacation Days = 10
